Cara Menggunakan Smart Lamp
1. Nyalakan ESP8266

Setelah program berhasil di-upload, nyalakan ESP8266 menggunakan USB atau adaptor 5V.
2. Sambungkan HP/Laptop ke WiFi ESP8266
Buka pengaturan WiFi di HP atau laptop.
Cari WiFi dengan nama:
SmartLamp
Sambungkan ke WiFi tersebut.
WiFi ini dibuat langsung oleh ESP8266, jadi tidak membutuhkan internet.
3. Buka Browser
Setelah tersambung ke WiFi SmartLamp, buka browser seperti:
Google Chrome
Microsoft Edge
Firefox
Browser bawaan HP
Lalu ketik alamat:
192.168.4.1
atau:
http://192.168.4.1
